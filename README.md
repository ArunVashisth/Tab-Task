TAB TASK PROJECT MANAGEMENT

This is a comprehensive task management platform designed to help teams organize projects and track progress in real time. It features a secure login system and a dynamic dashboard for efficient workflow management.

PROJECT FEATURES

1. Interactive Dashboard: View all your active projects and pending tasks at a glance.
2. Real Time Collaboration: Invite team members to join projects and assign specific tasks.
3. Task Organization: Categorize tasks by priority and track their status from start to completion.
4. Secure Authentication: Safe user login using Google OAuth and JWT tokens.
5. Media Uploads: Attach relevant files and images directly to your tasks.

TECHNICAL DETAILS

. Frontend: Built with React and styled using modern CSS.
. Backend: Powered by Node.js and Express for fast API responses.
. Database: Utilizes MongoDB for flexible and scalable data storage.
. Deployment: Backend hosted on Railway and Frontend on Vercel.

HOW TO RUN THE PROJECT

1. Install dependencies in both the frontend and backend folders.
2. Set up your environment variables for the database and authentication.
3. Use the start command to launch the server and the development client.
